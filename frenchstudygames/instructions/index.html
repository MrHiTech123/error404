How to run the game

Simply type the appropriate command into your command line:
Windows: no idea
Mac: python3 [filepath]
For [filepath], you should have every folder from your Users folder to your file separated by slashes. For example, if I had just finished downloading heephil.py into your downloads folder and wanted to run it, the filepath would be:

~/downloads/heephil.py
